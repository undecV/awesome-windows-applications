用過都說好的軟體推薦

undecV 2025-08-10T23:03:49+08:00

裝機必備的實用軟體大合集！

推薦一些 Windows 平台下的軟體， 適合日常以及非日常使用。
並不一定每一款軟體你都需要或適合你，也未必是大眾的選擇；
但是那些無時無刻不在彈廣告的辣雞軟體絕對不會是你的選擇。

覺得我廢話太多？
那就點我跳到索引！

本文的範圍是 應用軟體，不一定有加入例如 Google Docs、Spotify、Telegram 之類的 SaaS 網頁應用和服務， 想到會再加。以及像 Line 這種軟體我雖然在用，但我有的選嗎 :(

文章其中的一些軟體，我可能不常用，沒有追著最新版更新，甚至時下可能還沒有安裝，但我一定都有用過、體驗過。

這裡的選擇標準非常的...主觀。不是排名，不分先後。以下是我的選擇標準：

  • 確實能夠解決問題，且有效的解決問題；
  • 沒有騷擾性質的廣告；
  • 界面端莊優雅，易於使用。
  • 開源和自由軟體。

標記

本文的標記可能並不完全或有誤，甚至隨着時間的流逝而與現時不符，所以僅供參考，
若發現有漏標或錯標，又或是你有更好的選擇，歡迎告訴我 :)

  • 我們使用看上去十分專業的「雙名法」，即「發佈者／軟體名」。

  • 右側的 方便你貼到命令列中一鍵安裝（如果有對應的 軟體套件管理器 的話）。

  • 軟體的費用有不要錢的 免費， 還有課金變強的 加值， 和花錢才能用的 付費。

  • 軟體的授權有 自由 標記 開放原始碼 或 自由軟體；
    沒有使用上述開放授權的軟體用 專有 標記。

  • 標籤 全家桶 表示這是一個軟體集合，我都推薦（箱推）。

  • 標籤 硬核 表示這個軟體有較高的學習門檻。

  • 標籤 命令列工具 表示這個軟體要在命令列中使用。

  • 標籤 專業 表示使用這個軟體需要具備相關的專業知識。

  • 右側的例如 是連結到官方網站、對應的應用商店等連結。

                                              如何尋找優秀的實用軟體

                                              怎麼找？

                                              • 電腦的軟體的強大，不在於能做到多少事情，而是一件事情能做得多好。
                                              • 應用軟體應該旨在幫助用戶完成工作，提高效率。 而不是讓用戶的電腦成為廣告投放機。
                                              • 一些很棒的軟體功能非常強大，但是學習門檻較高，或是需要經過調教，甚至可能需要老司機帶路，但是學會了就能夠享受它們的便利了。
                                              • 不要小看網頁瀏覽器的附加元件＼擴充套件，他們能夠極大地提高體驗和便利。
